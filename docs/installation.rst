Installation
============

PyPi
----

**MetPyQC** is available on PyPI:

.. code-block:: bash

   pip install metpyqc

Future development will include MetPyQC on conda-forge channel.

Requirements
------------

Required dependencies are:

.. code-block:: bash

   numpy>=1.20.1
   scipy>=1.6.0
   pandas>=1.1.3
   scikit-learn>=0.23.2
   tqdm>=4.42.1
